# Estructuras de Datos – Trabajo Final

Este proyecto corresponde al trabajo final de la materia **Estructuras de Datos** y tiene como objetivo aplicar las distintas estructuras vistas durante la cursada a un dominio concreto y coherente. En particular, se modela un escenario en el que distintos países representan equipos de fútbol que disputan encuentros entre sí.

A lo largo del desarrollo se implementan diversas estructuras de datos, como árboles, pilas, colas y tablas hash, utilizándolas para organizar, almacenar y procesar la información del dominio. Los países se cargan dentro de una estructura de tipo árbol, donde se mantienen ordenados según un criterio lexicográfico, lo que permite realizar búsquedas y recorridos de manera eficiente.

Además, el sistema contempla la existencia de encuentros de fútbol entre los países, integrando las distintas estructuras para representar y gestionar estas relaciones. El proyecto busca reflejar no solo la correcta implementación de cada estructura, sino también su uso adecuado según el tipo de información y las operaciones requeridas.

Este trabajo permite consolidar los conceptos teóricos de la materia mediante su aplicación práctica, poniendo énfasis en la correcta elección de estructuras, la organización del código y la comprensión del comportamiento de cada una dentro de un problema real.
